# rpsGame
A generalized rock-paper-scissors CLI game with arbitrary number of moves and confirmation of the computer's move impartiality.

Project Structure

```
rpsGame/
│
├── index.js
├── utils/

│   └── validateArgs.js
├── models/
│   ├── Game.js
│   └── HelpTable.js
└── controllers/
    └── gameController.js
    ```
